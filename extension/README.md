# Frontned Logic

## Pipeline

1. User hovers over an image
2. Try on button appears
3. User clicks try on button
4. Send request to backend
5. Backend sends request to Replicate
6. Replicate sends request to Gemini Vision
7. Gemini Vision sends request to Replicate
8. Replicate sends request to backend
9. Backend sends request to extension
10. Extension displays try-on image
